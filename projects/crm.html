CRM Panel

Description

A custom-built CRM system for session and subscription management. This system allows administrators to create and manage clients, set booking slots, track session usage, send automated reminders, and visualize bookings on a calendar.

Tech Stack

  • Frontend: React, Tailwind CSS
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Features: Email notifications, dashboard stats, slot-based calendar
