CRM Panel

Description

This project is a modern, full-featured web-based CRM and appointment management system built for service-based businesses and organizations. The platform is designed to help admins, managers, and clients manage bookings, monitor session usage, communicate efficiently, and track key activities â€” all in one place.

Tech Stack

  • Frontend: React (TypeScript), Tailwind CSS, React Router, Axios, Socket.io, React Toastify
  • Backend: Node.js (Express), MongoDB (Mongoose)
  • Database: MongoDB
  • Export & Reporting: pdfkit, json2csv, docx for PDF/CSV/Word exports
  • Features: Email notifications, dashboard stats, slot-based calendar, WhatsApp API integration for appointment notifications
