Flying Packages

Description

A full-stack logistics web app to track deliveries in real-time. Features include package assignment, status updates, and an admin dashboard with insights.

Tech Stack

  • Frontend: React, CSS
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Features: Delivery status updates, admin analytics, role-based auth
